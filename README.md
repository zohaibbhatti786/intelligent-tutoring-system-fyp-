# intelligent-tutoring-system-fyp-
## Project Overview
An adaptive tutoring platform built with *Python and Django* that personalizes learning content based on student performance.
## Key Features
* *Adaptive Learning:* Suggests lessons based on quiz results.
* *Progress Tracking:* Real-time dashboard for students to monitor their growth.
* *NoSQL Database:* Uses *MongoDB* for flexible data storage.
## Tech Stack
* *Backend:* Python, Django
* *Database:* MongoDB
* *Frontend:* HTML5, CSS3
